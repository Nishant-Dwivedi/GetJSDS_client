Thank you for installing getJSDS.

GetJSDS is a light-weight extension that fetches some commonly-used data structures written in Javascript from an external server and copies them to your clipboard. You can start using the extension by following the steps mentioned below.

  1. Click on the toolbar button: from the menu to open the extension.
  2. Select a Data Structure from the dropdown menu.
  3. Copy the Data Structure to any online code editor by pressing Ctrl+V.
